# DAA-Group-2-Internal-Project
## HDB Resale Price Prediction
### This project explored the key factors that influence resale prices of HDB Flats in Singapore. We also developed a predictive machine learning model to estimate resale value based on property attributes.
### Our work followed the full end-to-end data analytics pipeline: Data Cleaning, Exploratory Data Analysis, Feature Engineering and Machine Learning Modelling
